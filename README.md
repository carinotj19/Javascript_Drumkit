# JavaScript Drum Kit

An interactive drum kit built using HTML, CSS, and JavaScript. Press designated keys to play different drum sounds.

## 📂 Files Included

- `index.html` – Main HTML file containing the structure of the drum kit.
- `style.css` – CSS file for styling the drum kit.
- `script.js` – JavaScript file handling the drum kit's functionality.
- `sounds/` – Directory containing audio files for each drum sound.
- `README.md` – Project documentation.

## 🚀 Getting Started

### Prerequisites

- A modern web browser (e.g., Chrome, Firefox, Edge).

### Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/carinotj19/Javascript_Drumkit.git
   ```

2. **Navigate to the project directory:**

   ```bash
   cd Javascript_Drumkit
   ```

3. **Open the application:**

   Open `index.html` in your preferred web browser to view the drum kit.

## 🛠️ Usage

Press the designated keys on your keyboard to play different drum sounds. Each key corresponds to a specific drum sound.
